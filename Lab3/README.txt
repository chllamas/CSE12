Lab 3: ASCII-risks (Asterisks)

-------------
Description

This lab is an introduction to MIPS ISA using MARS.  The 
program here has several nested loops to print an ASCII
diamond according to the input of user.  

------------
Files

- 
Lab3.asm

This file is the saved .asm from MARS that contains the source
code for the program.

------------
Instructions

The program can be opened using MARS.  In order to run, press 
the 'Assemble' button then hit 'Run'.  The program will prompt
for user input.  Integer must be given greater than 0.
